# Repository Instructions

This repository publishes the portable `project-due-diligence` Codex skill.

Keep the installable skill package under:

```text
project-due-diligence/
```

Do not add private project documents, diligence reports, workspace state, local machine paths, credentials, or customer materials to this repository.

When changing the skill:

- Update `project-due-diligence/SKILL.md`.
- Update `project-due-diligence/references/project-due-diligence-template.md` if the report structure changes.
- Bump the version in `SKILL.md` and `README.md` for material behavior changes.
- Keep examples portable and avoid absolute local paths.
